Example1:
-----------------
html
---------
<div class="box a">a</div>
<style>
  .box{
  width: 100px;
  height: 100px;
  background: #ddd;
  margin-bottom: 300px;
  position: relative;
}
</style>

---------
gsap code
---------
gsap.to(".a",{
x:400,
rotation:360,
duration:3
});
